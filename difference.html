What's the Difference?




HTML

Hypertext Markup language



Create the structure

-controls the layout of the content

-Provides the structure for web page design

-The fundamental building block of any web page

CSS

Cascading Style Sheet
Stylize the website

-Applies style to the website elements

-Targets Various screen size to make website responsive

-Primarily handles "tje look and feel" of a website

Javascript

Increase the Interactivity

-Add interactivity to a webpage

-Handles complex functions and features

-Programmatic code which enhances functionality
